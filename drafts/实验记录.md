## Mnist

20000样本，10类。

使用MMEncoder模型，如下配置。

```
class MnistParams(Params):
    n_patches = 8
    encoder_num = 1
    input_size = [784, 256]
    n_head = 4
    patches_d = 128
    hidden_d = 128
    class_num = 10
    pretrain_epoch = 50
    epochs = 100
    view_num = 2
    batch_size = 128
    lr = 3e-3
```

预训练50epoch，微调20epoch：acc：0.9975，nmi：0.99188，pri：0.9975。比较稳定
