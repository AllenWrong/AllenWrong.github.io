一、过拟合、欠拟合解决方案

1、一些概念

训练误差：在训练集上表现出的误差

泛化误差：表示着一个模型的泛化能力，通常使用测试集上的误差来进行近似泛化误差，当模型在测试集上的误差较小的时候，我们就认为这个模型的泛化能力比较强。因此，通过模型在测试集上的误差我们可以看出这个模型的泛化能力，进而确定是否使用这个模型。

验证集：测试集用于近似模型的泛化能力，通俗来讲就是用于评价模型是否能够被应用到真实场景中的大部分情况。因此超参数的选择就需要另外一部分数据，这就是验证集的作用。使用验证集进行调参，不同的超参数意味着不同的模型，这一步也是模型选择。

K折交叉验证：当数据集并不充分多的时候，就很难预留出大量的样本作为验证集，这时候会将原始训练集划分为$K$份，取其中的$K-1$份用于训练，用剩余的一份作为验证集，这样我们就会有进行$K$次训练和交叉验证，对这K次训练误差和验证误差取平均作为最终的训练误差和验证误差。

2、过拟合和欠拟合

模型无法得到较低的训练误差，我们将这一现象称作欠拟合（underfitting）。模型的训练误差远小于它在测试数据集上的误差，我们称该现象为过拟合（overfitting）。实际上，在实践中会有很多因素导致这两种拟合问题。这些关注了两种因素：模型复杂度好训练数据即集大小。

下面展示了 模型复杂度和误差之间的关系：

影响欠拟合和过拟合的另一个重要因素是训练数据集的大小。一般来说，如果训练数据集中样本数过少，特别是比模型参数数量（按元素计）更少时，过拟合更容易发生。此外，泛化误差不会随训练数据集里样本数量增加而增大。因此，在计算资源允许的范围之内，我们通常希望训练数据集大一些，特别是在模型复杂度较高时，例如层数较多的深度学习模型。

欠拟合又表现为高偏差，过拟合又表现为高方差。针对这种现象还有一些其他的解决方案：

欠拟合（又称为解决高方差）

  • 获得更多训练实例，也就是增加训练集
  • 减少特征的个数。类似于减小模型复杂度
  • 增加正则化程度。

过拟合（解决高偏差）

  • 增加特征的数量。类似于增加模型复杂度
  • 增加多项式特征。
  • 减小正则化程度。

丢弃法（官名 Dropout）

对于多层感知机，也就是神经网络而言，我们可以使用一定的概率来丢掉隐藏层中某些神经元，从而缓解过拟合问题。

二、梯度消失、梯度爆炸

  • 预训练技术可以缓解局部最优的问题
  • ReLU，批量标准化，残差网络可以缓解梯度消失的问题
  • L2正则化方法解决深度神经网络的过拟合并不合适，使用Dropout来缓解过拟合问题
  • GPU优化带宽，CPU优化延迟，所以GPU更适合做矩阵的运算

1、普适逼近定理

一个至少具有一层隐藏层的前馈神经网络，并且隐藏层包含有限数量的神经元（多层感知机），它可以以任意精度逼近一个定义在$R^n$闭集里的连续函数。前提是这个前馈神经网络的激活函数要满足一定的性质。比如Sigmoid函数，tanh函数，ReLU函数。

通俗来讲，就是说，一个至少具有一层隐藏层的前馈神经网络，并且隐藏层包含有限数量的神经元，它可以训练出该闭集里所有的模型。因此这样的神经网络就相当于这个闭集上的一个”元“，它可以生成该闭集内的任何一个函数。

2、梯度消失的解决方法

回顾Sigmoid激活函数的导数，它的范围是在$(0,0.25)$这个区间内，也就是说在反向传播的时候没经过一个层就会乘一个此区间内的数。由于这个数非常小，所以随着网络层次的增加，导数会慢慢的趋向于0，也就是梯度消失。所以Sigmoid函数并不适合做深度神经网络的激活函数。

一种常用的方法就是使用$ReLU$激活函数，$ReLU$激活函数的导数为0和1，所以在反向传播时，并不会出现梯度消失。另外$ReLU$可以被近似为$Softplus$函数。

使用深度残差网络。利用$ResNet$构造块构建深度残差网络。

Batch Normalization。批标准化主要的目的就是改变每个批量的原始分布，使他们都在相同的分布上。基本的方法就是批量内的每个样本减去该批量的平均值并除以标准差。

3、陷入局部最小的解决方法

预训练

随机初始化的方法本身就使得算法容易陷入局部最小值。随机初始化的参数与最优的参数可能相距甚远，因此随机初始化参数要到达最优参数就可能会经历许多局部最优参数。因此基于这个问题，就有预训练的方法，寻找一个好的网络初始化。限制玻尔兹曼机（RBM）是一种通过输入数据集学习概率分布的随机生成神经网络。所谓限制是指可见单元之间，隐藏单元之间没有相互连接。使用RBM单元构建深度信念网络（DBN）。DBN从结构上看就是把一个数据先压缩，然后再解压的过程。然后使用DBN的方式先把神经网络初始化到一个比较好的参数位置上，然后再把数据加进来做fine-tuning。

4、神经网络正则化

神经网络的正则，浅层神经网络使用$L2$正则化方法可能还比较有效，但是深层神经网络使用这种方法就是使$W$趋向于0，某个结点上产生的误差在整个深层神经网络中传播之后，在最后可能会造成巨大的误差。所以神经网络的正则化方法是Dropout。

所谓Dropout就是指训练时对网络中的神经元以一定的概率选择禁用，或者说是丢弃。注意：Dropout仅仅在训练阶段使用，在测试阶段所有的节点和连边都会被使用。

5、标签偏移

如果数据量足够的情况下，确保训练数据集和测试集中的数据取自同一个数据集，可以防止协变量偏移和标签偏移是正确的。如果数据量很少，少到测试集中存在训练集中未包含的标签，就会发生标签偏移。

三、循环神经网络进阶

1、GRU

下面是GRU单元的图示以及计算公式。


$$
R_{t} = σ(X_tW_{xr} + H_{t−1}W_{hr} + b_r)\
Z_{t} = σ(X_tW_{xz} + H_{t−1}W_{hz} + b_z)\
\widetilde{H}t = tanh(X_tW{xh} + (R_t ⊙H_{t−1})W_{hh} + b_h)\
H_t = Z_t⊙H_{t−1} + (1−Z_t)⊙\widetilde{H}_t
$$

  • 重置⻔有助于捕捉时间序列⾥短期的依赖关系；
  • 更新⻔有助于捕捉时间序列⾥⻓期的依赖关系。

def get_params():
    def _one(shape):
        # 按照正态分布随机初始化参数
        ts = torch.tensor(np.random.normal(0, 0.01, size=shape), device=device, dtype=torch.float32) #正态分布
        return torch.nn.Parameter(ts, requires_grad=True)
    def _three():
        return (_one((num_inputs, num_hiddens)),
                _one((num_hiddens, num_hiddens)),
                torch.nn.Parameter(torch.zeros(num_hiddens, device=device, dtype=torch.float32), requires_grad=True))

    W_xz, W_hz, b_z = _three()  # 更新门参数
    W_xr, W_hr, b_r = _three()  # 重置门参数
    W_xh, W_hh, b_h = _three()  # 候选隐藏状态参数

    # 输出层参数
    W_hq = _one((num_hiddens, num_outputs))
    b_q = torch.nn.Parameter(torch.zeros(num_outputs, device=device, dtype=torch.float32), requires_grad=True)
    return nn.ParameterList([W_xz, W_hz, b_z, W_xr, W_hr, b_r, W_xh, W_hh, b_h, W_hq, b_q])

def init_gru_state(batch_size, num_hiddens, device):   #隐藏状态初始化
    return (torch.zeros((batch_size, num_hiddens), device=device), )

GRU模型

def gru(inputs, state, params):
    W_xz, W_hz, b_z, W_xr, W_hr, b_r, W_xh, W_hh, b_h, W_hq, b_q = params
    H, = state
    outputs = []
    for X in inputs:
        # 上面的计算公式
        Z = torch.sigmoid(torch.matmul(X, W_xz) + torch.matmul(H, W_hz) + b_z)
        R = torch.sigmoid(torch.matmul(X, W_xr) + torch.matmul(H, W_hr) + b_r)
        H_tilda = torch.tanh(torch.matmul(X, W_xh) + R * torch.matmul(H, W_hh) + b_h)
        H = Z * H + (1 - Z) * H_tilda
        Y = torch.matmul(H, W_hq) + b_q
        outputs.append(Y)
    return outputs, (H,)

2、LSTM

长短期记忆long short-term memory:
遗忘门：控制上一时间步的记忆细胞
输入门：控制当前时间步的输入
输出门：控制从记忆细胞到隐藏状态
记忆细胞：⼀种特殊的隐藏状态的信息的流动

$$
I_t = σ(X_tW_{xi} + H_{t−1}W_{hi} + b_i) \
F_t = σ(X_tW_{xf} + H_{t−1}W_{hf} + b_f)\
O_t = σ(X_tW_{xo} + H_{t−1}W_{ho} + b_o)\
\widetilde{C}t = tanh(X_tW{xc} + H_{t−1}W_{hc} + b_c)\
C_t = F_t ⊙C_{t−1} + I_t ⊙\widetilde{C}_t\
H_t = O_t⊙tanh(C_t)
$$
长短期记忆模型在进行参数初始化的时候比GRU模型会有几部分不同。

长短期记忆模型从零实现，参考上面公式。

def lstm(inputs, state, params):
    [W_xi, W_hi, b_i, W_xf, W_hf, b_f, W_xo, W_ho, b_o, W_xc, W_hc, b_c, W_hq, b_q] = params
    (H, C) = state
    outputs = []
    for X in inputs:
        I = torch.sigmoid(torch.matmul(X, W_xi) + torch.matmul(H, W_hi) + b_i)
        F = torch.sigmoid(torch.matmul(X, W_xf) + torch.matmul(H, W_hf) + b_f)
        O = torch.sigmoid(torch.matmul(X, W_xo) + torch.matmul(H, W_ho) + b_o)
        C_tilda = torch.tanh(torch.matmul(X, W_xc) + torch.matmul(H, W_hc) + b_c)
        C = F * C + I * C_tilda
        H = O * C.tanh()
        Y = torch.matmul(H, W_hq) + b_q
        outputs.append(Y)
    return outputs, (H, C)
