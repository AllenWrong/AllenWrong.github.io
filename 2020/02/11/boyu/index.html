5、Softmax

在数学，尤其是概率论和相关领域中，Softmax函数，或称归一化指数函数，是逻辑函数的一种推广。它能将一个含任意实数的K维向量Z“压缩”到另一个K维实向量sigmoid z中，使得每一个元素的范围都在（0,1）之间，并且所有元素的和为1(也可视为一个 (k-1)维的hyperplan，因为总和为1，所以是subspace)。

6、torch.tensor.view()

def softmax(X):
    # 分子
    X_exp = X.exp()

    # 分母
    partition = X_exp.sum(dim=1, keepdim=True)
    return X_exp / partition

3.Softmax回归模型
$$
\begin{aligned} \boldsymbol{o}^{(i)} &= \boldsymbol{x}^{(i)} \boldsymbol{W} + \boldsymbol{b},\ \boldsymbol{\hat{y}}^{(i)} &= \text{softmax}(\boldsymbol{o}^{(i)}). \end{aligned}
$$

# X represente features
def nex(X):
    # input_nums represent the input number of the input net
    return softmax(torch.mm(X.view(-1,input_nums), W) + b)

4.损失函数

对交叉熵损失函数优化效果的通俗理解：

交叉熵公式：
$$
H\left(\boldsymbol y^{(i)}, \boldsymbol {\hat y}^{(i)}\right ) = -\sum_{j=1}^q y_j^{(i)} \log \hat y_j^{(i)},
$$
交叉熵损失函数公式：
$$
\ell(\boldsymbol{\Theta}) = \frac{1}{n} \sum_{i=1}^n H\left(\boldsymbol y^{(i)}, \boldsymbol {\hat y}^{(i)}\right ),
$$
Log函数是一个增函数，对于$log(\hat{y})$，y_hat是一个位于0-1之间的数值，在0-1之间log取值为负数。简单看一个例子：实际标签$[0,0,1]$与预测值$[0.1,0.2,0.9]$的乘积实际上就是log(0.9)，所以交叉熵结果就是对实际所属类别处的预测值取log()。这样我们只需要看log函数在0-1之间的曲线，这是一个增长曲线，然而，按正常思维，我们希望损失越小越好，所以要加个负号，这样预测的越准确，损失就会越小。概括的来说，交叉熵就是拿到当前所属类别索引出的预测值，然后取log()。这个操作和之前提到的gather()函数的操作是一样的。只不过我们需要清楚地知道预测值矩阵和实际标签是什么样的。下面是一个简单的列子：

两个样本的预测输出矩阵：
$$
[[0.1, 0.8, 0.3, 0.2]\ [0.3, 0.2,0.1, 0.9]]
$$
实际标签：
$$
y=[1,3]
$$
实际标签中指出了当前实际所属类别的索引，因此我们应根据实际标签进行取值，pytorch实现是这样的：

y_hat = torch.tensor([[0.1, 0.8, 0.3, 0.2], [0.3, 0.2,0.1, 0.9]])
y = torch.tensor([1,3])
print(y_hat.gather(1, y.view(-1,1)))
"""
tensor([[0.8000],
        [0.9000]])
"""

基于上一步，取log然后再取负就可实现交叉熵。下面是pytorch实现的交叉熵，通过交叉熵可以对gather()函数有很好的理解。

def cross_entropy(y_hat, y):
    return - torch.log(y_hat.gather(1, y.view(-1, 1)))

5.定义准确率

def accuracy(y_hat, y):
    return (y_hat.argmax(dim=1) == y).float().mean().item()

y_hat.argmax(dim=1) == y再行方向上取去最大值的索引，也就是取出预测的分类，然后判断是否跟实际标签相等，但是这个结果是个布尔值，所以使用float()要把他转换成数值，然后对本次所有样本的取平均值作为本次的预测准确度，通过item()取出其中的数值。下面是个演示例子：

demo_hat = torch.tensor([[0.1,0.2,0.9,0.2],[0.2,0.9,0.2,0.1],[0.2,0.1,0.9,0.1]])
demo_y = torch.tensor([3, 1, 2])
print((demo_hat.argmax(dim=1) == demo_y).float())
print((demo_hat.argmax(dim=1) == demo_y).float().mean())
print((demo_hat.argmax(dim=1) == demo_y).float().mean().item())

封装一下，有下面的代码，表示模型net在数据集data_iter上的准确度：

def evaluate_accuracy(data_iter, net):
    acc_sum, n = 0.0, 0
    for X, y in data_iter:
        acc_sum += (net(X).argmax(dim=1) == y).float().sum().item()
        n += y.shape[0]
    return acc_sum / n

# 引包
import torch
from torch import nn
from torch.nn import init
import numpy as np

import sys

import d2lzh1981 as d2l  # 主要使用了里面的一些函数，可以使用其他的包替代

num_input = 784
num_output = 10

class LinearNet(nn.Module):
    # Pytorch所有的网络结构都要基础nn.Module模块
    # 这只是一种实现方式，我习惯使用这种
    def __init__(self, num_input, num_output):
        super(LinearNet, self).__init__()
        self.linear = nn.Linear(num_input, num_output)

        y = self.linear(x.view(x.shape[0], -1))
        return y

loss = nn.CrossEntropyLoss()

optimizer = torch.optim.SGD(net.parameters(), lr=0.01)

# 训练。可以使用其他的训练函数进行训练。

d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, batch_size, None, None, optimizer)

1、关于激活函数

  1. ReLu激活函数

    ReLu函数提供的是一种非常简单的非线性变化，从定义可以看出，该函数把负数清零，保留正数元素。
    $$
    \text{ReLU}(x) = \max(x, 0).
    $$
    ReLu函数的实现和图像：（在Pytorch中可以使用torch.nn.ReLu()调用ReLu函数

    def relu(X):
    return torch.max(input=X, other=torch.tensor(0.0))

     # 这是一个绘图函数，可以查看函数的图像
 def xyplot(x_vals, y_vals, name):
        # 把x和y转成ndarray再进行绘制
     plt.plot(x_vals.detach().numpy(), y_vals.detach().numpy())
     plt.xlabel('x')
     plt.ylabel(name + '(x)')

  2. Sigmoid函数

    在二分类中常用的激活函数，效果非常好。sigmoid函数可以将元素的值变换到0和1之间，下面是定义：

    $$
    \text{sigmoid}(x) = \frac{1}{1 + \exp(-x)}.
    $$
    下面是使用numpy实现的sigmoid函数。在Pytorch中可以通过torch.nn.sigmoid()进行sigmoid运算。

    import numpy as np
def sigmoid(x):
    return 1 / (1 + np.exp(-1 * x))

  1. tanh（双曲正切）

    函数可以将元素的值变换到-1和1之间：
    $$
    \text{tanh}(x) = \frac{1 - \exp(-2x)}{1 + \exp(-2x)}.
    $$

    当输入接近0时，tanh函数接近线性变换。虽然该函数的形状和sigmoid函数的形状很像，但tanh函数在坐标系的原点上对称。下面是使用numpy实现的tanh函数。在Pytorch中可以通过：

    def tanh(x):
    x_exp = np.exp(-2 * x)
    return (1-x_exp)/(1+x_exp)
    # return (np.exp(x) - np.exp(-x)) / (np.exp(x) + np.exp(-x))

x = np.array([-8., 0., 8.])
print(tanh(x))

  2. 在Pytorch中对张量调用激活函数

    在一开始，我以为使用下面的这种方式：

    a = torch.tensor([-8., 0., 8.])
m = torch.nn.Tanh(a)
print(m)

    而实际上应该使用这种方式进行调用

    a = torch.tensor([-8., 0., 8.])
r = torch.nn.ReLU()
s = torch.nn.Sigmoid()
t = torch.nn.Tanh()
print(r(a))
print(s(a))
print(m(a))

  3. 关于激活函数的选择

  • ReLu函数是一个通用的激活函数，目前在大多数情况下使用。但是，ReLU函数只能在隐藏层中使用。
    • 用于分类器时，sigmoid函数及其组合通常效果更好。由于梯度消失问题，有时要避免使用sigmoid和tanh函数。
  • 在神经网络层数较多的时候，最好使用ReLu函数，ReLu函数比较简单计算量少，而sigmoid和tanh函数计算量大很多。
    • 在选择激活函数的时候可以先选用ReLu函数如果效果不理想可以尝试其他激活函数。

2、多层感知机实现

所谓多层感知机就是至少包含一个隐藏层的由全连接层组成的神经网络。

什么是隐藏层？一个网络结构由输入层（输入层一般没有什么特别的作用）、隐藏层和输出层组成，因此一个网络结构除了输入层和输出层之外就是隐藏层了，输出层是最后一次，输入层是第一层。

什么是全连接层？全连接层中的每一层都是许多神经元组成的平铺结构，这是全连接层最显著的特征，以后还会遇到其他各种各样的网络层。

因此在实现多层感知机的时候，步骤基本上还是和上面是一样的， 不同的只是网络结构的定义和参数的维度。

从零开始实现多层感知机

  1. 数据集获取和初始化模型参数

    # 引包
import torch
import numpy as np
import sys
sys.path.append("/home/kesci/input")
import d2lzh1981 as d2l
print(torch.__version__)

# 获取数据即集
batch_size = 256
train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size,root='/home/kesci/input/FashionMNIST2065')

# 设置输入层、输出层、隐藏层神经元的个数
num_inputs, num_outputs, num_hiddens = 784, 10, 256

# 进行参数的随机初始化。这里要注意W1、W2的形状。
# 这里的网络结构是：
#     - 第0层：输入层
#     - 第1层：隐藏层
#     - 第2层：输出层
# 一般来说第0层，称为输入层，W1的行数等于第0层的神经元的个数，列数等于第1层神经元的个数
# Wn 行数：第n-1层的神经元的个数；列数：第n层的神经元的个数
# 数值：初始化的W符合正态分布，初始化的b为0
W1 = torch.tensor(np.random.normal(0, 0.01, (num_inputs, num_hiddens)), dtype=torch.float)
b1 = torch.zeros(num_hiddens, dtype=torch.float)
W2 = torch.tensor(np.random.normal(0, 0.01, (num_hiddens, num_outputs)), dtype=torch.float)
b2 = torch.zeros(num_outputs, dtype=torch.float)

# 
params = [W1, b1, W2, b2]
for param in params:
    param.requires_grad_(requires_grad=True)

# 定义激活函数
def relu(X):
    return torch.max(input=X, other=torch.tensor(0.0))

  2. 定义网络结构

    这个网络结构中，首先将每个样本拉直，也就是每个样本数据都转换成了一行数据。然后进行隐藏层的计算，并使用ReLU函数进行处理。然后又进行了输出层的运算。实际上网络的结构在模型参数初始化哪里体现的最明显。

    def net(X):
    X = X.view((-1, num_inputs))
    H = relu(torch.matmul(X, W1) + b1)
    return torch.matmul(H, W2) + b2

  3. 训练

    # 设置训练周期和学习率
num_epochs, lr = 5, 100.0
# 调用训练器进行训练
d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, batch_size, params, lr)

num_inputs, num_outputs, num_hiddens = 784, 10, 256

net = nn.Sequential(
        # FlattenLayer的作用就是把样本拉直
        d2l.FlattenLayer(),
        # 使用ReLU的隐藏层
        nn.Linear(num_inputs, num_hiddens),
        nn.ReLU(),
        # 输出层
        nn.Linear(num_hiddens, num_outputs), 
        )  
# 这定义了模型结构，这是与之前那种方式不同的一种实现。可以看出，这种方式更加直观

for params in net.parameters():
    init.normal_(params, mean=0, std=0.01)

# 定义损失函数
loss = torch.nn.CrossEntropyLoss()

# 定义优化函数。这里传入了模型参数和学习率，在进行训练的时候就不用传参了
optimizer = torch.optim.SGD(net.parameters(), lr=0.5)

# 调用训练器进行训练
num_epochs = 5  # 定义训练周期为5
d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, batch_size, None, None, optimizer)
